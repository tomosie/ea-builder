Cara Pasang EA di MT5 Anda

1

Klik tombol Unduh .MQ5 di panel atas.

2

Di MT5 Anda, buka Tools -> MetaQuotes Language Editor (atau tekan tombol F4).

3

Tarik/Seret file .mq5 yang Anda unduh ke folder MQL5 \ Experts di navigator sebelah kiri.

4

Buka berkas di MetaEditor, klik tombol Compile di bar atas (atau pencet F7). Selesai!
